============
Installation
============


Install pipelib from PyPi:

.. code-block:: console

    $ pip install pipelib


or development from the code:

.. code-block:: console

    $ git clone https://github.com/vsoch/pipelib
    $ cd pipelib
    $ pip install -e .


Next check out the :ref:`getting_started-user-guide` pages for more detail to use the library.
